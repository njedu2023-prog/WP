# WP V2 人工决策辅助

## 目标与边界

WP V2 用于帮助人工在 14:20 至收盘期间只选择一支主票，并在不满足条件时明确等待或空仓。系统不连接券商，不读取账户，不创建订单，也不执行买卖。

## 决策链

1. 仅保留主板且涨跌停规则不高于 10% 的候选；创业板、科创板、北交所股票不进入尾盘交易候选。
2. 计算市场广度、强弱股比例、涨跌停数量和尾盘候选数量，输出 `允许寻找机会`、`谨慎`、`回避` 或 `数据不足`。
3. 14:20 后持续保留仍具内在资格的观察票，并记录连续合格次数、连续领先次数和质量排名变化。
4. 对每支候选输出 T+1 开盘、最高、最低、收盘的 Q10/Q50/Q90 经验区间，以及盈利、触及 +3%、触及 -3% 和触及涨停概率。
5. 14:30 前默认等待。14:30 后只有市场环境、稳定性、领先幅度、预测置信度和风险收益全部通过，才输出 `建议关注买入`；14:50 仍不通过则输出 `建议空仓`。
6. T+1 根据人工策略生成卖出建议：触及 -3% 风险线时建议风险退出；10:40 仍未封住涨停时建议人工择机卖出；涨停封稳则继续观察至收盘，确认未炸板后提示 T+2 09:31 人工评估。

## 预测口径

- `实时样本校准`：真实 14:20-14:50 主票快照已达到配置门槛。
- `混合先验`：真实快照不足，同时使用低权重日线代理样本。
- `日线代理先验`：只有历史收盘日线代理，仅作方向参考，不宣称为真实尾盘校准。
- `样本不足`：不输出数值建议，并阻止建议买入。

所有训练样本必须早于预测日，防止未来数据泄漏。当前只能预测 T+1 四价区间和触及概率；“先涨后跌”的分钟级路径顺序，要等 T+1 分时快照积累后再建模。

## 收益解释

预测区间和概率是历史条件分布，不是收益保证。页面中的买入和卖出措辞均为人工复核建议；实际成交价格、仓位、滑点、交易费用和是否持仓由使用者自行确认。
