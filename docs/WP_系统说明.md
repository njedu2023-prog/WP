# WP 系统说明

WP 用于筛选今日涨幅超过 6%、前一日未涨停、今日未涨停的 A 股股票，并按次日涨停概率排序输出 Top50。第一阶段采用规则评分模型，后续可接入历史校准和机器学习模型。

当前主流程：

1. 从 `a-share-top3-data/data/wp/latest/wp_latest_rank_input.csv` 读取候选输入。
2. 硬过滤：今日涨幅超过 6%、前一日未涨停、今日未涨停、成交额达到配置门槛。
3. 计算强板块、强个股、资金承接、动量、资金、形态、风险惩罚。
4. 输出 `p_limitup_t1`、`wp_score`、信号等级、核心理由和风险提示。
5. 生成 CSV、JSON、HTML，并由 GitHub Pages 发布。
6. 支持手动历史区间回测，输出 Top10/20/50、AUC、Brier、次日涨幅和回撤指标。

实时模式会检查行情日期。如果上游 latest 行情日期不是当天交易日，报告显示“数据日期过期”，避免把旧行情误当成当天收盘数据。
